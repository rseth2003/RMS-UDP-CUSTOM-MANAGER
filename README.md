# RMS UDP CUSTOM MANAGER v2.0
By Rodney Mwase Seth

## Features
- Full custom installation (no original scripts)
- UDP Custom tunnel for HTTP Custom app
- Create users with Minutes / Hours / Days expiry
- Auto-expiry daemon checks every 5 seconds
- Expired users auto-deleted and kicked immediately
- Beautiful colored menu with icons

## Requirements
- Ubuntu 20.04 / 22.04 x86_64
- Root access

## Install
```bash
sudo -i
git clone https://github.com/rseth2003/RMS-UDP-CUSTOM-MANAGER && cd RMS-UDP-CUSTOM-MANAGER && chmod +x install.sh && ./install.sh
```

## Usage
```bash
udp
```
